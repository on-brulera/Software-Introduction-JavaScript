Constantes

Permite crear variables que no tendran cambio durante la ejecución del programa y se recomienda escribirlo con letras mayúsculas y siempre debe tener un valor al declararlo.

Para valores Primitivos

A estos valores (string, number, boolean, null, undefined, NaN) se los accede directamente y por ello una vez declarada una constante no es posible cambiarlo. 

        
            const PI = 3.1419;
            console.log("Constante: ", PI); //salida: "Constante 3.1416"
            PI = 23.23 //salida: Error

Para valores compuestos

A estos valores: object={}, array[], function(){}, Class{} y otros al ser accedidos por una dirección y no a los valores directamente permiten modificar el valor de la constante 

                                        
            //crear objeto
            let persona = {
                nombre: "Henry",
                edad: 20
            }
            console.log(persona) //salida: Object nombre, edad
            persona.correo = "henry@gmail.com";
            console.log(persona); //salida: Object nombre, edad, correo

            
            //crear lista
            let colores = [
                "blanco", "azul", "rojo"
            ]
            console.log(colores); //salida: Array(3)
            colores.push("verde");
            console.log(colores); //salida: Array(4)
