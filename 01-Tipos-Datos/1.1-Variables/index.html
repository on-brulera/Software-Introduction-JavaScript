Variables

var

Las variables declaradas con este término son de ámbito global, pero pueden ser modificadas dentro de bloques, funciones y condicionales (poco recomendable)

Ejemplo

                var global = "HOLA MUNDO GLOBAL";
                console.log("global antes del bloque: ", global); // salida: "HOLA MUNDO GLOBAL"
                {
                    var global = "HOLA MUNDO GLOBAL DENTRO";
                    console.log("global dentro del bloque: ", global); //salida: "HOLA MUNDO GLOBAL DENTRO"
                }
                console.log("global despues del bloque: ", global); //salida: "HOLA MUNDO GLOBAL DENTRO"

let

Las variables declaradas con este término son de ámbito de bloque, por ello no se ven afectadas si en otras funciones tienen el mismo nombre de variable (recomendable)

Ejemplo

                let bloque = "HOLA MUNDO BLOQUE";
                console.log("var_bloque antes del bloque: ", bloque); //salida: "HOLA MUNDO BLOQUE"
                {
                    let bloque = "HOLA MUNDO BLOQUE DENTRO";
                    console.log("var_bloque dentro del bloque: ", bloque); //salida: "HOLA MUNDO BLOQUE DENTRO"
                }
                console.log("var_bloque despues del bloque: ", bloque); // salida: "HOLA MUNDO BLOQUE"
